## 🚀 Como Usar

### 1️⃣ **Adicionar a Foto**
- Coloque a foto dos noivos com o nome **`image.jpg`** na mesma pasta do site
- A imagem deve estar em proporção 3:4 (melhor: 500x750 pixels)
- Formatos: JPG, PNG, WEBP

### 2️⃣ **Rodar o Site Localmente**

#### **Opção A: PHP Server (Recomendado)**
Abra o terminal/PowerShell e rode:
```bash
cd "C:\Users\ESGC\Desktop\J&L site"
php -S localhost:8000
```
Depois acesse: **http://localhost:8000**

#### **Opção B: Duplo clique no index.html**
Basta abrir o arquivo `index.html` direto no navegador.

#### **Opção C: XAMPP/WAMP**
- Coloque a pasta em `htdocs` (XAMPP) ou `www` (WAMP)
- Acesse: `http://localhost/J&L site/`
